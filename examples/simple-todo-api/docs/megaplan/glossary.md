# Glossary

Canonical domain terms for this project. Every agent session uses these words — no synonyms, no improvisation.

## Terms

| Term | Definition | Canonical example | Common confusions |
|------|------------|-------------------|-------------------|
| Todo | An item representing a task to be completed, containing a title and completion status. | `{"id": 1, "title": "Buy milk", "completed": false}` | Synonyms: task |
| User Account | Credentials and profile of an individual registered user. | `{"id": 1, "username": "alice"}` | Synonyms: profile |
